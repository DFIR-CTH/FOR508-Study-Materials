# Windows Event Log Types

## 1. Security
- Records access control and security settings
- Events based on audit and group policies
- **Examples**: Failed logon, folder access

## 2. System
- Contains events related to Windows Services, system components, drivers, resources, etc.
- **Examples**: Service stopped, system rebooted

## 3. Application
- Software events unrelated to operating system
- **Examples**: SQL server fails to access a database, antivirus alert

## 4. Custom Application and Services
- Custom application logs
- **Examples**: Task Scheduler, Terminal Services, PowerShell, WMI, Firewall, DNS (Servers)
